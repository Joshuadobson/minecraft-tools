Minecraft Map Art Generator

Convert any image into a Minecraft map art layout using accurate perceptual colour matching. Upload a photo or design, crop it to a square, and the tool instantly maps every pixel to the closest available Minecraft block colour.

Features

Supports 128×128 single map output — the standard size for one Minecraft map item. Multi-map grid export lets you split a large image across a 2×2, 3×3, or 4×4 grid of maps for large-scale builds. Floyd-Steinberg dithering improves colour accuracy in gradients and complex images. Download your result as a PNG, or export block counts as CSV or JSON to plan your materials.

How it works

The tool uses CIELAB colour space matching to find the closest Minecraft block for each pixel — more accurate than simple RGB comparison. All processing happens in your browser, no data is uploaded to any server.
